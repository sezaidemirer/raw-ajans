# Raw Ajans - Yedek Kopya

## Yedekleme Tarihi
**5 Kasım 2025**

## İçerik
Bu klasör, Raw Ajans projesinin tam bir yedeğini içermektedir.

### Yedeklenen Dosyalar:
- `src/` - Tüm React bileşenleri, stiller ve i18n dosyaları
- `public/` - Public assets
- `index.html` - Ana HTML dosyası
- `package.json` & `package-lock.json` - Bağımlılıklar
- `vite.config.js` - Vite konfigürasyonu
- `tailwind.config.js` - Tailwind CSS konfigürasyonu
- `postcss.config.js` - PostCSS konfigürasyonu
- `.github/` - GitHub Actions workflow dosyaları (varsa)
- Diğer konfigürasyon dosyaları

### Not:
- `node_modules/` klasörü yedeklenmedi (npm install ile tekrar oluşturulabilir)
- `.git/` klasörü yedeklenmedi (git repository)

### Geri Yükleme:
1. Bu klasörün içeriğini proje dizinine kopyalayın
2. `npm install` komutu ile bağımlılıkları yükleyin
3. `npm run dev` ile geliştirme sunucusunu başlatın
